# Innonymous - anonymous chat
The project is completed as System and Network Administration assignment.
### Usage
```
git clone https://github.com/smthngslv/sna-final.git
cd sna-final
cp .env.example .env
docker-compose up -d
```

### Access
* http://localhost:8080 WEB
* http://localhost:8000 API

### Requirements
* docker-compose v3.9+
* docker v19.03+
